# AstrBot 督工与更新预告插件

这个插件提供四类能力：

1. 给自己挂督工任务。
2. 在群聊里 `@别人` 给对方挂督工任务。
3. 目标用户发言时，按冷却时间触发催工提醒。
4. 每日固定时间发送“更新内容”和“内容预告”。

本插件的提醒文案支持优先调用当前会话使用中的大模型，并按当前 AstrBot 人格或自定义人格要求生成催工话术；如果模型不可用，会自动回退到本地模板。

## 督工命令

```text
督工 开始 任务=写第一章 待办=大纲、正文、校对 时长=3h 冷却=2h 抽取=3
督工 开始 @小明 任务=做海报 待办=出图、排版 时长=2h 冷却=1h
督工 状态 [@目标]
督工 完成 [@目标]
督工 取消 [@目标]
督工 帮助
```

参数说明：

1. `任务=` 必填。
2. `待办=` 可选，用顿号、逗号或换行分隔。
3. `时长=` 可选，例如 `30m`、`2h`、`1d`。
4. `冷却=` 可选，例如 `10m`、`2h`，冷却内不会重复催。
5. `抽取=` 可选，每次提醒展示几条待办。

## 播报命令

```text
更新内容 设置 时间=21:00 内容=今天更新了第一章和封面
更新内容 开
更新内容 关
更新内容 状态
更新内容 立即发送

内容预告 设置 时间=20:00 内容=明天预告第二章和设定图
内容预告 开
内容预告 关
内容预告 状态
内容预告 立即发送
```

## 设置页添加任务

插件配置里有 `设置页督工任务` 列表，可以直接新增任务。命令创建的进行中任务也会同步显示到这里；任务完成、取消或过期后会自动从列表移除。

私聊任务最少填写：

```text
目标用户ID=QQ号
任务=写第一章
待办=大纲、正文、校对
时长=3h
冷却时间=2h
```

群聊任务还需要填写 `群号`，或直接填写完整 `会话ID`：

```text
napcat:GroupMessage:群号
```

如果不填写完整会话ID，插件会按 `平台ID`、`会话类型`、`目标用户ID` 或 `群号` 自动生成。默认平台ID是 `napcat`。

## 大模型人格催工

如果启用了 `llm_enabled`：

1. 插件会优先使用当前会话对应的聊天模型生成催工话术。
2. 如果开启了 `llm_follow_active_persona`，会把当前 AstrBot 会话人格一并带入。
3. 还可以用 `llm_custom_prompt` 追加专门的督工人格要求。

例如可以把 `llm_custom_prompt` 配成：

```text
说话像盯稿的编辑，语气凶一点，短句，不卖萌，不要太客气，但不要辱骂。
```

## 已知限制

1. 当前版本的督工提醒只在目标用户于原会话发言时触发，不会独立主动轮询骚扰。
2. 一个用户同一时间只允许存在一个进行中的督工任务。
3. “更新内容”和“内容预告”当前是“每日固定时间 + 固定文案”的实现。
